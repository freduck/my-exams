To run this application you must ensure you have the following installed on you PC 
1) Node.js version 16 and above
2) mongo DB at the time of writing this application i used version 8.2

   download and extract the folder to your desktop
   cd into the folder ensure you are connected to the internet  you can open your terminal and type 
   NPM INSTALL to install all the necessary modules then  type 
   nodemon app.js
   the open any browser and type 
   a) localhost:3000 to view home page
   b) localhost:3000/start-exam to take exam
